# About HS5

HS5 (High-performance Scale-up Self-hosted Simple Storage Service) is an open-source object storage solution designed for high performance and simplicity. It is API-compatible with AWS S3 and is ideal for users who want to self-host their storage on a single node.

- **High performance**: Optimized for speed and efficiency.
- **Scale-up**: Designed for single-node deploymentsâ€”scale by upgrading your hardware.
- **Self-hosted**: You control your data and deployment.
- **Simple**: Easy to set up and maintain.
- **S3-compatible**: Works with existing S3 tools and clients.

HS5 is freely available under the [LGPLv3+ license](https://github.com/uroni/hs5#LGPL-3.0-1-ov-file).

For more details, see the [README](../index.html) or visit the [GitHub repository](https://github.com/uroni/hs5).
